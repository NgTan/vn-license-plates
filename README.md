# Vietnam License Plates

A node.js module to look up province by various license plates

# Installation

```
	$ npm install vn-license-plates
```

# Usage
```js
  var licensePlates = require('vn-license-plates');
  
  licensePlates.lookup('59'); // Ho Chi Minh
```
